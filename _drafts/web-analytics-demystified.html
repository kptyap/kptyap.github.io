  • Kaizn is one of the main principles of web analytics - small incremental improvements over time rather than large one-offs
  • Evolution of analytic sources from web server logs to client side javascript tags
  • Main disadvantages of client side:
    1. Dependence on cookies
    2. Implementation of customised tagging
    3. Unable to collect certain data types eg. file downloads
  • Useless terms: Hit, Click-Through (use response rate)
